# Processamento Digital de Sinais

Neste repositório encontram-se todos os materiais utilizados na disciplina de Processamento Digital de Sinais, ministrado durante os anos de 2019 até 2026 na UFERSA - Campus Pau dos Ferros.

## Assuntos Abordados no Curso

1. Introdução ao Processamento Digital de Sinais
2. Sinais de Tempo Discreto
3. Sistemas de Tempo Discreto
4. Transformada de Fourier em Tempo Discreto
5. Amostragem
6. Transformada $Z$
7. Transformada $Z$ Inversa
8. Propriedades da Transformada $Z$
9. Análise no Domínio da Frequência de Sistemas LTI
10. Sistemas de Fase Mínima
11. Discretização de Sistemas de Tempo Contínuo
12. Realização de Sistemas de Tempo Discreto
13. Estruturas de Implementação para Sistemas IIR
14. Estruturas de Implementaçào para Sistemas FIR
15. Especificação de Projeto de Filtros Digitais
16. Projeto de Filtros Digitais FIR
17. Projeto de Filtros Digitais IIR
18. Série Discreta de Fourier
19. Transformada Discreta de Fourier
20. Convolução Circular
21. Implementação Computacional da Transformada Discreta de Fourier
22. Efeitos da Discretização em Sistemas de Processamento Digital de Sinais

5. LYONS, Richard G. Understanding Digital Signal Processing, 3th Edition, Pearson, 2010.
